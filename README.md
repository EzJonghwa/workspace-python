## 프로젝트 환경설정

1. 저장소 복제
```
    git clone https://github.com/EzJonghwa/workspace-python.git
```

2. 가상환경 생성 (예 : 가상환경 이름 myenv)
```
cd workspace-python
python -m venv myenv

```
3. 가상환경 활성화
```
    .\myenv\Scripts\Actvate
```

4. 필수 라이브러리 설치
```
    pip install -r requirments.txt
```
